Como o Sistema Funciona:

    Interface Intuitiva: O sistema apresenta uma interface limpa e moderna com explicação sobre a Lei de Benford.

    Fontes de Dados:

        População de países (via API REST Countries)

        Preços de ações (dados simulados)

        Magnitudes de terremotos (dados simulados)

        Dados personalizados (gerados para seguir a Lei de Benford)

    Funcionalidades:

        Busca dados de APIs públicas

        Extrai o primeiro dígito significativo de cada número

        Compara a distribuição observada com a distribuição esperada pela Lei de Benford

        Exibe gráficos comparativos

        Calcula métricas de conform
